# 🔐 Decodificador de Texto - Challenge ONE

Este é um projeto de criptografia e descriptografia de texto desenvolvido durante o programa Oracle Next Education (ONE) em parceria com a Alura. O foco principal foi o desenvolvimento de lógica de programação e manipulação de strings.

## 🚀 Demonstração
Você pode testar a aplicação online aqui: https://decodificador-de-texto-psi-one.vercel.app/

## 🛠️ Funcionalidades
- **Criptografar:** Converte vogais em códigos específicos:
  - `a` -> `ai`
  - `e` -> `enter`
  - `i` -> `imes`
  - `o` -> `ober`
  - `u` -> `ufat`
- **Descriptografar:** Retorna o texto codificado para a língua original.
- **Copiar:** Botão funcional para copiar o resultado para a área de transferência.

## 💻 Tecnologias Utilizadas
- **HTML5:** Estruturação dos elementos da página.
- **CSS3:** Estilização responsiva e uso de variáveis (Custom Properties).
- **JavaScript:** Lógica de substituição de caracteres e manipulação do DOM.

## 📝 Aprendizados
Este projeto foi fundamental para consolidar conceitos que hoje utilizo nos meus estudos de **Backend Python**, como:
1. Controle de fluxo e iteração (loops).
2. Manipulação de índices em strings.
3. Versionamento de código com Git.
4. Deploy contínuo via Vercel.
